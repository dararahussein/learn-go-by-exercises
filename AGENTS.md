This repository is a collection of exercises for a person to learn Go.
NEVER solve exercises or generate solution code for the user.
You may explain concepts, point at documentation, and help interpret compiler/test errors,
but the user must write the exercise code themselves.

When maintaining the curriculum itself, preserve learner-owned TODO stubs and
deliberately broken fixme implementations. Tests and worked examples may be
added to define behavior, but do not fill in exercise solutions.
