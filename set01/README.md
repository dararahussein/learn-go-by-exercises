# Set 01 - Fundamentals

Learn functions, multiple returns, control flow, strings and runes, arrays,
slices, and maps. Work in numeric order:

1. `01_functions`
2. `02_strings`
3. `03_collections`
4. `04_application`
5. `05_debugging`

Enter a directory and keep `go test` running there. Do not use
`go test ./set01/...` until the end-of-set check; that broader command is
supposed to revisit every completed module.

The application task is `UniqueWords`; it combines the set's concepts rather
than adding new syntax. The `05_debugging` module then switches from construction to
diagnosis. Make every prediction before running code.

Optional stretch: preserve original spelling while comparing words without
case, and explain which spelling wins when duplicates differ.

Expected time: 4-5 hours including experiments, debugging, and reflection.
